Untuk bisa menjalankan web ini ikuti langkah berikut:

1. Pastikan sudah menginstall node.js di komputer anda
2. Buka folder Projek-ef di VS Code
3. Buka terminal di VS Code
4. Jalankan perintah "npm install" untuk menginstall dependencies INI PENTING
5. Jalankan perintah "npm run dev" untuk menjalankan web
